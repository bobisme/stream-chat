# Stream Chat TUI

- usage: `streamchat <youtube-url>` to join a live stream or premier chat
- use the latest version of ratatui (had some API changes recently, so consult docs)
- `cat tui.txt` to see ansi-colored design.
- Type in the text input, enter to send.
- ctrl+j/ctrl+k scroll down/up in the chat by 10 lines (if there is content to scroll back to)
